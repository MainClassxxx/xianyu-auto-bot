## 🎨 **前端风格规范**

### 颜色方案（简约浅色系 + 可爱卡通）
```css
--primary: #FFB6C1;      /* 浅粉 */
--secondary: #87CEEB;    /* 浅蓝 */
--success: #98FB98;      /* 浅绿 */
--warning: #FFE4B5;      /* 浅橙 */
--danger: #FFC0CB;       /* 粉红 */
--background: #F5F7FA;   /* 浅灰 */
--card-bg: #FFFFFF;      /* 白色 */
--text: #606266;         /* 深灰 */
--border: #E6E6E6;       /* 边框 */
```

### 设计元素
- 圆角：8px-16px
- 阴影：柔和 0 2px 12px rgba(0,0,0,0.1)
- 图标：Emoji + 卡通 SVG
- 动画：流畅 0.3s ease
- 字体：圆润可爱
