## 🚀 使用指南

### 1. 启动服务

**后端**:
```bash
cd /Users/macxiaoli/.openclaw/workspace/xianyu-auto-bot
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8080
```

**前端**:
```bash
cd frontend
npm run dev
```

### 2. 访问界面

- **前端管理**: http://localhost:3000
- **API 文档**: http://localhost:8080/docs
- **健康检查**: http://localhost:8080/health

### 3. 配置闲鱼账号

1. 访问闲鱼网页版：https://goofish.com/
2. 登录账号
3. F12 → Network → 复制 Cookie
4. 在前端管理界面添加账号

### 4. 配置自动发货

1. 进入"自动发货"页面
2. 添加规则：
   - 规则名称：教程自动发货
   - 关键词：教程
   - 发货内容：链接：https://example.com
   - 库存：-1（无限）

### 5. 查看日志

```bash
tail -f data/bot.log
```
