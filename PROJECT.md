## 🚀 升级计划

### 第一阶段：核心功能（本周）

1. **完善闲鱼 API 客户端**
   - 完整的 Cookie 管理
   - WebSocket 实时消息
   - 错误重试机制

2. **多账号管理**
   - 账号 CRUD
   - 账号状态监控
   - 账号隔离

3. **商品管理**
   - 商品列表获取
   - 商品上下架
   - 商品编辑（含改价）

4. **订单管理**
   - 订单列表
   - 订单状态同步
   - 发货操作

5. **对话消息**
   - 消息列表
   - 发送消息（文字/图片）
   - 会话管理

### 第二阶段：自动化功能（下周）

6. **自动回复系统**
   - 规则管理（关键词、商品匹配）
   - AI 回复集成
   - 回复历史记录

7. **自动发货系统**
   - 规则管理（多规格、库存）
   - 自动执行
   - 发货记录

8. **通知系统**
   - 多渠道（飞书、Telegram、钉钉、企业微信）
   - 事件触发
   - 通知记录

### 第三阶段：高级功能（后续）

9. **自动评价**
10. **自动免拼**
11. **AI 知识库**
12. **数据统计**
13. **发货流程编辑器**
