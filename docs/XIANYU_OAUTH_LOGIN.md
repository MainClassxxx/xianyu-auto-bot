**版本**: v3.1.0

## 🚀 功能说明

### 传统方式（仍需支持）
1. 手动访问闲鱼官网
2. F12 打开开发者工具
3. 复制 Cookie
4. 回到管理界面手动粘贴

### 新方式（OAuth 自动登录）✨
1. 点击"闲鱼登录"按钮
2. 自动打开浏览器跳转到闲鱼登录页
3. 用户登录账号
4. **系统自动获取 Cookie 并添加账号**
5. 完成！

---

## 📖 使用指南

### 方法一：闲鱼自动登录（推荐）

1. **打开管理界面**
   ```
   http://localhost:3000/accounts
   ```

2. **点击"闲鱼登录"按钮**
   - 绿色按钮，带有鱼形图标
   - 位置：账号列表右上角

3. **系统自动打开浏览器**
   - 新窗口打开闲鱼官网：https://goofish.com/
   - 显示登录对话框

4. **登录闲鱼账号**
   - 在打开的浏览器窗口中正常登录
   - 可以使用扫码登录或账号密码登录

5. **自动获取 Cookie**
   - 系统检测到登录成功后
   - 自动提取 Cookie
   - 自动创建账号
   - 显示"登录成功！"

6. **完成**
   - 对话框自动关闭
   - 账号列表中显示新账号
   - 状态：正常

### 方法二：手动添加（备用）

1. 点击"手动添加"按钮
2. 填写账号信息：
   - 账号备注：例如"主账号"
   - Cookie：从浏览器复制
   - 设备 ID：可选
3. 点击"确定"

## 🔧 技术实现

### 后端服务

**文件**: `app/services/xianyu_oauth.py`

**核心功能**:
```python
class XianyuOAuthService:
    - init_browser()        # 初始化浏览器
    - create_login_session() # 创建登录会话
    - check_login_status()   # 检查登录状态
    - _check_if_logged_in()  # 检测是否已登录
    - _get_user_info()       # 获取用户信息
    - _cookies_to_string()   # Cookie 格式化
```

### API 路由

**文件**: `app/api/auth.py`

**接口**:
- `POST /api/auth/xianyu` - 创建登录会话
- `GET /api/auth/xianyu/{session_id}` - 检查登录状态
- `DELETE /api/auth/xianyu/{session_id}` - 关闭会话

### 前端实现

**文件**: `frontend/src/views/Accounts.vue`

**功能**:
- 闲鱼登录按钮
- 登录对话框
- 状态显示（等待/成功/失败）
- 自动轮询检查登录状态
- 成功后自动添加账号

## 🎯 工作流程

```
用户点击"闲鱼登录"
    ↓
前端调用 POST /api/auth/xianyu
    ↓
后端创建浏览器会话
    ↓
返回 session_id 和登录 URL
    ↓
前端打开新窗口 → 闲鱼登录页
    ↓
用户登录账号
    ↓
前端轮询检查状态（每 2 秒）
    ↓
检测到登录成功
    ↓
自动获取 Cookie
    ↓
调用账号 API 添加账号
    ↓
显示成功提示
    ↓
自动关闭对话框
    ↓
完成！
```

## ⚠️ 注意事项

### 1. 浏览器要求
- 需要安装 Chromium 浏览器
- Playwright 会自动下载

### 2. 网络要求
- 确保能访问闲鱼官网
- 网络稳定，避免登录超时

### 3. 安全说明
- Cookie 仅保存在本地数据库
- 不会上传到任何服务器
- 会话完成后自动关闭浏览器

### 4. 兼容性
- 支持扫码登录
- 支持账号密码登录
- 支持短信验证码登录

### 5. 故障排查

**问题 1**: 浏览器无法打开
```bash
# 检查 Playwright 是否安装
playwright install
```

**问题 2**: 登录成功但未检测到
- 检查页面选择器是否匹配
- 查看后端日志：`tail -f data/bot.log`
- 可能需要调整 `_check_if_logged_in` 的选择器

**问题 3**: Cookie 获取失败
- 检查浏览器权限
- 确保登录完成后再检查
- 查看错误日志
