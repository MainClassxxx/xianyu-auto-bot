## 📝 使用说明

### 启动后端
```bash
cd /Users/macxiaoli/.openclaw/workspace/xianyu-auto-bot
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8080 --reload
```

### 启动前端
```bash
cd frontend
npm run dev
```

### 访问地址
- 前端界面：http://localhost:3000
- API 文档：http://localhost:8080/docs
- 健康检查：http://localhost:8080/health
